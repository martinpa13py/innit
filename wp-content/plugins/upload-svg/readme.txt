Safely enable SVG uploads with sanitization and prevent XML/SVG vulnerabilities on your WordPress website. Preview SVG files in your Media Library.

== Description ==

Upload SVG is a secure and user-friendly WordPress plugin that allows you to safely upload and manage SVG files in your Media Library. SVG files can potentially contain malicious code, but with Upload SVG, you can enable automatic sanitization to prevent XML/SVG vulnerabilities on your website. This plugin ensures your SVG uploads are handled with enhanced security while offering a seamless user experience.

### Features

*	Securely upload SVG files to your Media Library.
*	Enable automatic sanitization to protect against XML/SVG vulnerabilities.
*	Effortlessly manage and preview SVG files in your Media Library.
*	Restrict SVG uploads to trusted user roles for additional security.
*	Seamless integration with the latest version of WordPress.
*	Lightweight and optimized for optimal performance.

SVG Sanitization is implemented using the following library: [https://github.com/darylldoyle/svg-sanitizer](https://github.com/darylldoyle/svg-sanitizer).

== Installation ==

1. Install through the WordPress directory or download, unzip, and upload the "upload-svg" folder to your /wp-content/plugins/ directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to "Upload SVG" to configure the plugin options.
4. Enjoy safely managing SVG files on your WordPress website!

== Frequently Asked Questions ==

= Can I upload SVG files without sanitization? =

Yes, you can choose to enable or disable the automatic sanitization option based on your requirements. However, we strongly recommend enabling it for enhanced security.

= Can I limit SVG uploads to specific user roles? =

Absolutely! Upload SVG allows you to restrict SVG uploads to trusted user roles, ensuring that only authorized users have the permission to upload SVG files.
